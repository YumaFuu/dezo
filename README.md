# Dezo
Dezo is a cli tool that transrates Japanese between English.


## Installation

    $ gem install dezo

## Usage

`$ dezo`


and then type a word which you wanna transrate.


## Depends on
Dezo depends on デ辞蔵Webサービス

- デ辞蔵Webサービス
https://www.est.co.jp/dev/dict/REST
